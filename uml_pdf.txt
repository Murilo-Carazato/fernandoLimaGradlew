Exercício 3
A loja “Musica S/A”, especializada na venda de CDs antigos, deseja
implementar um software para fazer o controle de suas vendas.
De acordo com o gerente da loja, cada disco deverá ser cadastrado
individualmente, permitindo ao usuário relacioná-lo com seu gênero e
respectivo artista. O software deverá ainda disponibilizar uma tela para
cadastro de clientes que além dos dados básicos, deve ainda permitir que o
usuário relacione o gênero e artista preferido do cliente, caso existam.
Além dos cadastros iniciais descritos anteriormente, o software deverá
disponibilizar uma tela para cadastrar as vendas onde basicamente, o usuário
deverá relacionar o cliente, o vendedor e os discos que o cliente está
comprando.
Em questão de relatórios, o software deverá disponibilizar para o usuário um
relatório de clientes por gênero, clientes por artista e vendas por período.
Com base na descrição apresentada, implemente um diagrama de classes
que contemple as necessidades da loja.
